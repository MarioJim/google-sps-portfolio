Current projects

A music client for touchégg

One thing I really missed from using Windows was the support for gestures on my laptop's trackpad, so about a month ago I found this program called touchégg that collected the gestures detected by the trackpad's driver and not only executed some action of you choosing, but also showed some cool animations.

I really liked the idea, so I'm making an app that receives the gestures and displays the current volume and information about the music currently playing.
